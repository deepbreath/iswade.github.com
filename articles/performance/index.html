    perf 工具使用

    通过两个函数 foo, bar 各占50%的CPU使用率进行perf工具示例演示。

    示例程序

    #include <stdio.h>
#include <stdlib.h>
int max = 5000000;

int foo() {
    int i = 0;
    long long sum = 0;
    for (i = 0; i < max; i++)
        sum += i;
    printf("foo %ld\n", sum);
}

int bar() {
    int i = 0;
    long long sum = 0;
    for (i = 0; i < max; i++)
        sum += i;
    printf("bar %ld\n", sum);
}

int main() {
    while (1) {
        foo();
        bar();
    }
}

    直接使用 perf top 可以查看系统全局所有使用CPU较高的函数 

    # perf top
...
             samples  pcnt function                    DSO
             _______ _____ ___________________________ ___________________

               21.00 15.2% do_page_fault               [kernel.kallsyms]  
               17.00 12.3% find_job                    /lib64/ld-2.11.3.so
               14.00 10.1% copy_page                   [kernel.kallsyms]  
               13.00  9.4% bgp_delete                  /lib64/ld-2.11.3.so
                9.00  6.5% dl_main                     /lib64/ld-2.11.3.so
                7.00  5.1% _dl_start                   /lib64/ld-2.11.3.so
...
    可以看出内核函数do_page_fault使用率较高。

    记录 CPU 占用率

    # perf record -p  88653

    生成火焰图

    git clone https://github.com/brendangregg/FlameGraph
cd FlameGraph
perf record -F 99 -a -g  -p 114819
perf script | ./stackcollapse-perf.pl > out.perf-folded
./flamegraph.pl out.perf-folded > perf-main.svg
