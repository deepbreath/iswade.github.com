    gdb命令参考

    基础命令

    gdb programNam programParas handle SIG35 SIGTERM SIGUSR2 nostop noprint nopass ignore set args programParas start r bt watch *(void *)0x7fffb2bbd483 p st_array[12].index watch &(st_array[12].ucNumber) + 2 watch (void *)((char *)(&(st_array[12])) + 129 + 96 + 24) set can-use-hw-watchpoints 0 detach q

    b funName command bt c end c

    set i=0 while (i < 20) print array[$i] set $i++ end

    call funName("name")

    ptype structName

    p *((int *)(ebp)) p *((int *)(ebp + 4)) p *((int *)(ebp + 8)) p *((int *)(ebp + 12))

    调试coredump文件

    gdb ./a.out a.coredump bt

    反汇编

    在定位死机问题时，经常会用到定位死机所在位置，通常用bt可以打印出死机所在位置，我们可以进一步用disassemble命令查看反汇编代码定位所在行。 gdb programName disassemble 0x02388232 # 反汇编地址

    通常情况下，我们不同版本可能对系统的底层函数有不同的实现，比如想对比两个版本的差异就可以通过反汇编函数来比较 disassemble function_name # 反汇编函数

    watch

    定位踩内存时，经常会用到的一个命令，比如监控一个地址什么时候被写，可以使用下面命令打印出调用栈： watch *(void *)0x7fffb2bbd483 bt
